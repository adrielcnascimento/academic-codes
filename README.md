## 📁 Descrição das Pastas

### `Atividades - Micro`
Atividades relacionadas à disciplina de Microcontroladores, cobrindo programação embarcada e periféricos.

### `C-cpp`
Implementações em C e C++ desenvolvidas ao longo do curso, incluindo estruturas de dados, algoritmos e programação de sistemas.

### `Códigos práticas STM32 - M&M`
Códigos desenvolvidos nas aulas práticas de Microcontroladores e Microprocessadores (M&M) utilizando a plataforma STM32.

### `Disciplinas Eng Comp - IFPB`
Repositório central com os projetos e atividades das diferentes disciplinas do curso de Engenharia de Computação do IFPB.

### `python`
Scripts Python desenvolvidos para exercícios, automações e projetos acadêmicos.

### `systemverilog`
Projetos e módulos em SystemVerilog para simulação e síntese de circuitos digitais.

### `thehuxley`
Soluções dos problemas resolvidos na plataforma [The Huxley](https://thehuxley.com/), utilizada em disciplinas de programação.
